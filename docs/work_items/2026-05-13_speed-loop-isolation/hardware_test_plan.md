# 速度环隔离测试计划

## 测试元数据

- 日期：2026-05-13
- 作者：AI agent
- 功能：速度环隔离测试命令 `WSleft,right`
- 固件构建：待填写
- 硬件版本：STM32F446 小车当前实物

## 目标

确认在绕过位置环和角度环后，左右轮速度 PID 是否仍会导致低速抖动、PWM 大幅摆动或左右轮响应明显不一致。

## 前置条件

- 小车已重新理线，编码器和电机线束固定。
- 测试区域后方至少有 1 m 安全距离；本轮优先倒车，不做前进长距离测试。
- 操作者手边保留急停能力，可立即发送 `S` 或断电。
- 串口连接参数：`COM15`，`961200` baud。

## 命令序列

先做离地或架空短测，确认命令方向和停止有效：

1. `S`
2. `R0`
3. `WS-0.02,-0.02`
4. 连续发送 `O` 观察 3 到 5 次
5. `S`
6. `O`

地面低速倒车测试：

1. `S`
2. `R0`
3. `WS-0.03,-0.03`
4. 每 0.2 到 0.5 秒发送一次 `O`，持续 3 到 5 秒
5. `S`
6. `O`

如果低速稳定，再做稍高速倒车测试：

1. `S`
2. `R0`
3. `WS-0.05,-0.05`
4. 每 0.2 到 0.5 秒发送一次 `O`，持续 3 到 5 秒
5. `S`
6. `O`

可选左右轮单独测试，建议架空进行：

1. `S`
2. `R0`
3. `WS-0.03,0.00`
4. 连续发送 `O`
5. `S`
6. `WS0.00,-0.03`
7. 连续发送 `O`
8. `S`

## 期望行为

- `O` 输出中 `mode=SPEED_TEST`。
- `CTRLDBG` 中 `pos_err=0.000`、`ang_err=0.00`，说明位置环和角度环没有参与。
- `CTRLDBG l_sp/r_sp` 应等于 `WS` 设置的左右轮目标速度。
- `ODOM ls/rs` 应逐步接近目标速度，符号一致，抖动幅度可观察但不应出现持续大幅反向跳变。
- PWM 不应在小目标速度下长期打满或左右剧烈交替。

## 失败信号

- `mode` 没有变成 `SPEED_TEST`。
- `S` 后电机没有停止。
- `ODOM ls/rs` 与目标速度符号长期相反。
- 小目标速度下 PWM 长期接近上限，或左右 PWM 剧烈跳变。
- 车体在无角度环时仍明显左右摆动。

## 安全停止 / 恢复

- 停止命令：`S`
- 状态复位：`R0`
- 必要时直接断电，再检查电机线、编码器线、地线和供电。
