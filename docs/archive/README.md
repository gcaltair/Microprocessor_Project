# Archive Index

This directory contains historical, superseded, or supporting documents that should remain available for traceability but are no longer part of the primary AI-agent read path.

## Rules

- Archived documents are read only when a current index explicitly points to them.
- Archived documents do not override current status, priorities, or workflow rules.
- If an archived document contains still-useful material, copy the needed conclusion into a current document instead of silently promoting the archive back to active status.
